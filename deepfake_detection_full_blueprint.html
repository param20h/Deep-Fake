Step-by-step roadmap
P1

Setup & data — Day 1–2

Colab + Kaggle setup. Download DFDC Preview Dataset (lighter subset, ~10GB) or FaceForensics++ via Kaggle API. Install: torch, torchvision, facenet-pytorch (MTCNN), opencv-python, gradio. Verify T4 GPU active.

P2

Face detection pipeline — Day 2–3

Load MTCNN from facenet-pytorch. Extract + align faces from all training images. For videos: sample every 10th frame using OpenCV. Save cropped faces as preprocessed dataset (saves training time).

P3

Model fine-tuning — Day 3–5

Load EfficientNet-B4 pretrained on ImageNet. Replace final classifier layer: Linear(1792, 1) + Sigmoid. Freeze early layers, train last 3 blocks + classifier. Loss: BCEWithLogitsLoss. Optimizer: AdamW, lr=1e-4, weight decay=1e-5. Target: ~93%+ AUC.

P4

Video analysis module — Day 5–6

Build video_predict() function: open with cv2.VideoCapture → sample N frames (N=15–20) → run face detect + classify each frame → return mean score + confidence + per-frame breakdown. Flag as fake if mean score > 0.5.

P5

Heatmap visualization — Day 6–7

Implement Grad-CAM on EfficientNet. Overlay heatmap on face crop showing "why the model thinks it's fake". Saves as PNG. This is the #1 feature recruiters will notice — shows you understand model interpretability.

P6

Gradio web app — Day 7–8

Build tabbed interface: Tab 1 = image upload → face crop + verdict + Grad-CAM heatmap. Tab 2 = video upload → frame-by-frame score + final verdict. Add confidence bar. Deploy to HuggingFace Spaces via GitHub push.

P7

Browser extension — Day 8–10

Chrome MV3 extension: right-click any image on a webpage → "Check if deepfake" → sends image URL to your FastAPI backend → shows popup with real/fake score. See extension section below for full file structure.

File structure (Chrome MV3)

4 files total. No framework needed — plain JS.

01
manifest.json — declares permissions: contextMenus, activeTab, scripting. Sets popup.html as default popup.
02
background.js — registers right-click context menu on images. On click, sends image src to content.js.
03
content.js — receives image URL, calls your FastAPI endpoint (fetch POST), injects result overlay badge on the image.
04
popup.html — simple UI showing last checked image + score. Optional but looks polished in screenshots.

Challenges to expect

Dataset size — Full DFDC is 470GB. Use the DFDC Preview (6GB) or FaceForensics++ compressed version on Kaggle. More than enough for a portfolio model.

No face detected — MTCNN may miss faces in low-res or angled images. Add a fallback: if no face found, resize full image to 224×224 and classify anyway.

Overfitting — EfficientNet is powerful, can memorize training data. Use dropout=0.4, heavy augmentation (flip, color jitter, cutout), and early stopping.

Extension CORS issues — Browser extensions can't fetch image pixels cross-origin. Send the image URL to your backend; let the server download and process it instead.
